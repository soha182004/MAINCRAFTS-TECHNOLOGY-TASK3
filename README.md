## 📌 Overview

This repository contains the RTL design, testbenches, and simulation results for four fundamental **sequential logic circuits**, implemented and verified using **Verilog HDL** on **EDA Playground** (Verilator simulation engine, EPWave waveform viewer).

This was completed as **Task 3** of a VLSI Design Internship, building on combinational logic design (Task 2) to sequential/clocked circuits — the foundation of registers, counters, memories, and finite state machines used in real digital systems and processors.
